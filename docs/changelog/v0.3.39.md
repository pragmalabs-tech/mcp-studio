# 0.3.39

### Features

**Edit test name**
- Tests can now be renamed directly from the test card. Click the name to edit it inline without leaving the current view.

### Distribution

**npx support**
- MCP Studio is now available on npm as `@pragmalabs/mcp-studio`. Run it without any installation: `npx @pragmalabs/mcp-studio`.

**Homebrew tap**
- Added a Homebrew tap for macOS users: `brew tap pragmalabs-tech/mcp-studio && brew install mcp-studio`. The formula is automatically updated on every release.
