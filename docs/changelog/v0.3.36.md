# 0.3.36

### Features

**Snapshot center**
- Introduced a centralized `SnapshotCenter` that manages widget snapshots independently of the action recording flow. Snapshots are now keyed by widget frame ID rather than passing around live iframe references, making the system more resilient when iframes are re-created between steps.

**Fullscreen preview for snapshots**
- Step review now includes a fullscreen preview mode for snapshots. Click to expand any captured snapshot to full screen for a closer inspection of the widget state at that point in the recording.

**Per-action snapshots for widget interactions**
- Widget click, canvas click, and text input actions now each capture a snapshot at the moment of interaction. These snapshots are stored and surfaced in the history drawer and replay result dialog so you can review the exact visual state at every step.
